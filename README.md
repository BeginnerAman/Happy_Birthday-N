# 🎉 Happy Birthday Neha

An interactive, beautifully animated celebration experience featuring dynamic confetti, photo gallery, background music, and personalized voice messages to make special moments truly unforgettable.

## ✨ Features

- **Animated Confetti**: Dynamic particle effects with physics simulation that trigger on page load and interactions
- **Photo Gallery**: Auto-rotating slideshow displaying cherished moments with captions
- **Music Playback**: Celebratory background music with play/pause controls
- **Voice Message**: Personalized audio greeting with progress tracking and playback controls
- **Floating Decorations**: Animated balloons, hearts, and stars for festive ambiance
- **Smooth Animations**: CSS-driven transitions, gradient backgrounds, and sparkle effects
- **Responsive Design**: Fully optimized for desktop and mobile devices
- **Accessibility**: ARIA labels, semantic HTML, keyboard navigation (Escape key to close)

## 🛠️ Tech Stack

- **HTML5** - Semantic markup with accessibility support
- **CSS3** - Modern animations, gradients, and responsive layouts (49.1%)
- **Vanilla JavaScript** - Interactive controls and canvas animations (29.4%)
- **Canvas API** - Confetti particle system with realistic physics
- **HTML5 Audio API** - Audio playback with custom progress bars

## 🎮 How to Use

1. **Open the page** - Confetti animation plays automatically with gradient background
2. **Play Music** - Click "🎵 Play Birthday Song" button to toggle music
3. **Listen to Message** - Click the special message button to open the personalized greeting
4. **View Photos** - Gallery auto-advances every 3.5 seconds with smooth transitions
5. **Close Popup** - Click ✖ or press Escape key to close

## 🎨 Technical Highlights

- **Confetti System**: 140+ particles on initial burst, gentle 12-particle loop every 1.6 seconds
- **Canvas Optimization**: Responsive sizing with efficient requestAnimationFrame loop
- **Mobile Optimization**: Touch event handling for audio preload on mobile devices
- **Audio Preload**: Optimized with `rel="preload"` for faster playback
- **Performance**: CSS transforms and opacity for 60fps animations
- **Accessibility**: Full ARIA support with keyboard navigation
